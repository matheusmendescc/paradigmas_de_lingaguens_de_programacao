Projeto de Microserviços

A proposta é de desenvolver a demonstração de uma aplicação implementada utilizando o estilo arquitetônico baseado em objetos e estilo arquitetônico de microserviços.

Serviços

Serviço de aplicacao

http://127.0.0.1:5000 {% if application %} ON {% else %} OFF {% endif %}

Serviço de usuarios

http://127.0.0.1:5001 {% if usuarios %} ON {% else %} OFF {% endif %}

Serviço de postagens

http://127.0.0.1:5002 {% if postagens %} ON {% else %} OFF {% endif %}

Serviço de comentarios

http://127.0.0.1:5003 {% if comentarios %} ON {% else %} OFF {% endif %}

Endpoints do Serviço de usuários

Carregando a lista de todos os usuários:

GET "/usuarios"

Carregando um usuário pelo id:

GET "/usuarios/{string:user_id}"

Carregando o perfil completo de um usuário pelo id.

GET "/usuarios/{string:user_id}/perfil"
Esse endpoint retorna o perfil do usuário com a adição das postagens feitas.

Criando um usuário.

POST "/usuarios/"
BODY {"username":str, "nome":str, "bio":str}

Atualizando um usuário.

PUT "/usuarios/{string:user_id}/"
BODY {"bio":str}

Removendo um usuário.

DELETE "/usuarios/{string:user_id}"

Endpoints do Serviço de postagens

Carregando a lista de todas as postagens:

GET "/postagens"

Carregando uma postagem pelo id:

GET "/postagens/{string:postagem_id}"

Carregando postagens por usuário.

GET "/postagens/usuario/{string:user_id}"
Esse endpoint retorna todas as postagens do usuário com o user_id.

Criando uma postagem.

POST "/postagens/"
BODY {"uid":str,"conteudo":str}

Atualizando uma postagem.

PUT "/postagens/{string:postagem_id}/"
BODY {"conteudo":str}

Removendo uma postagem.

DELETE "/postagens/{string:postagem_id}"

Endpoints do Serviço de comentarios

Carregando a lista de todos os comentarios:

GET "/comentarios"

Carregando um comentario pelo id:

GET "/comentarios/{string:comentario_id}"

Carregando comentarios por postagem.

GET "/comentarios/postagem/{string:postagem_id}"
Esse endpoint retorna todos os comentarios de uma postagem com o postagem_id.

Criando um comentario.

POST "/comentarios/"
BODY {"uid":str,"pid":str,"conteudo":str}

Atualizando um comentario.

PUT "/comentarios/{string:comentario_id}/"
BODY {"conteudo":str}

Removendo um comentarios.

DELETE "/comentarios/{string:comentario_id}"
